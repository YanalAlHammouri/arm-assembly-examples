# arm-assembly-examples
A collection of ARM assembly implementations of fundamental algorithms and programming concepts, including sorting, searching, , array processing, and subroutines.

## Included Programs

### Basic Operations
- Addition
- Multiplication
- Even or odd detection
- Sum from 1 to N
- Factorial

### Array Processing
- Array creation and memory access
- Array sum
- Minimum and maximum values
- Counting values greater than a threshold
- Loading and storing values with `LDR` and `STR`

### Subroutines
- Greater-value function
- Array-sum function

### Algorithms
- Linear Search
- Fibonacci sequence
- Decimal-to-binary conversion
- Bubble Sort

## Concepts Demonstrated

- ARMv7 assembly
- Register manipulation
- Conditional branching
- Loops
- Memory addressing
- `LDR` and `STR` instructions
- Arrays
- Functions and subroutines
- Searching and sorting algorithms
